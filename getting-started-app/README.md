# Node.js DevOps Project

## Project Description
This is a Node.js application containerized using Docker.

## Technologies Used
- Node.js
- Git
- GitHub
- Docker
- Docker Compose
- Jenkins 
- AWS EC2 

## How to Run

1. Clone the repository
2. Run:
   docker compose up -d
3. Open:
   http://localhost:3000
